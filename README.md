# Boka Limo Services

Custom WordPress website for private transportation services.

Features:
- Custom Route Post Type
- Route Search
- Autocomplete
- Booking Form
- Email Notifications
- Multilingual (EN/ME)
- Responsive Design

Tech Stack:
- WordPress
- PHP
- JavaScript
- CSS
- Polylang
