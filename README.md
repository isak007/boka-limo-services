
# Boka Limo Services

Boka Limo Services is a custom WordPress website built for a private transportation company.

The project includes route management, route search, multilingual support, and a booking system that sends customer inquiries directly via email.

Features:
- Custom Route Post Type
- Route Search
- Autocomplete
- Booking Form
- Email Notifications
- Multilingual (EN/ME)
- Responsive Design

Tech Stack:
- WordPress
- PHP
- JavaScript
- CSS
- Polylang

Challenges

- Implemented custom route search using WP_Query and custom fields.
- Built multilingual support using Polylang.
- Created a custom JavaScript autocomplete component.
- Implemented email booking requests with server-side validation.
